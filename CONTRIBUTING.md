# Contributing to RenText

Thank you for your interest in contributing! 🎉

## Ways to contribute

- **Bug reports** — Found something wrong? Please [open an issue](../../issues/new?template=bug_report.md).
- **Feature requests** — Have an idea? Please [open an issue](../../issues/new?template=feature_request.md).
- **Pull requests** — Code contributions are welcome.

## Before submitting a pull request

1. Fork the repository and create a branch from `main`.
2. Make sure the project builds without errors: `dotnet build`
3. Keep changes focused — one feature or fix per PR.
4. Describe what you changed and why in the PR description.

## Build instructions

**Requirements:** .NET 9 SDK

```bash
git clone https://github.com/umineko73/RenText.git
cd RenText
dotnet run
```
