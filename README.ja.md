# RenText

**テキストエディタ感覚で使えるファイルリネームツール**

フォームに入力したりルールを設定したりする必要はありません。ファイル名を1行1ファイルとして直接編集するだけです。

## 主な機能

- **テキストエディタ感覚の編集** — ファイル名を1行ずつ直接編集できます
- **リアルタイムプレビュー** — 変更前後を右ペインで確認（`変更前 → 変更後`）
- **VS Code スタイルの検索・置換** (`Ctrl+H`) — 正規表現・大文字小文字区別に対応
- **正規表現プレビュー** — 入力しながらリアルタイムに結果を確認
- **完全ロールバック付き保存** (`Ctrl+S`) — 1件でもエラーがあればすべて元に戻す
- **ソート機能** — ファイル名・更新日・拡張子（昇順/降順）
- **表示形式切替** — ファイル名のみ / 更新日時付き / サイズ付き / 日時＋サイズ
- **フォルダツリー** — 遅延読み込み・ドライブ種別アイコン対応
- **セッション復元** — 最後に開いたフォルダを記憶し、次回起動時に自動で開く（ツリーも自動展開）
- **コマンドライン引数対応** — フォルダパスを引数に渡すと、そのフォルダを開いて起動
- **ダーク / ライトモード** 切替
- **キーボードナビゲーション** — ファイル一覧で `↑` `↓` `PageUp` `PageDown`

## 使い方

### 基本的な流れ

1. 左のフォルダツリーまたはアドレスバーでフォルダを選択
2. 中央ペインでファイル名を直接編集
3. 右のプレビューペインで変更内容を確認
4. `Ctrl+S` で一括保存

### キーボードショートカット

| キー | 操作 |
|------|------|
| `Ctrl+S` | 保存（リネーム実行） |
| `Ctrl+H` | 検索・置換パネル 開閉 |
| `F5` | フォルダツリーとファイル一覧を手動リフレッシュ |
| `↑` / `↓` | ファイル間の移動（カーソル列位置を維持） |
| `PageUp` / `PageDown` | 1ページ分移動 |

### 検索・置換

- `Ctrl+H` で検索・置換パネルを開く
- **正規表現**・**大文字小文字区別**に対応
- 入力中にプレビューペインへリアルタイム反映

#### 正規表現の使用例

`()` でファイル名の一部をキャプチャし、置換欄で `$1`・`$2`・… として参照できます。

| 検索 | 置換 | 変換前 | 変換後 |
|------|------|--------|--------|
| `^\d+[-_\s](.+)` | `$1` | `001_photo.jpg` | `photo.jpg` |
| `(.+)_(.+)\.jpg` | `$2_$1.jpg` | `family_2024.jpg` | `2024_family.jpg` |
| `(\d{4})-(\d{2})-(\d{2})` | `$1$2$3` | `photo_2024-05-21.jpg` | `photo_20240521.jpg` |
| `\s*[\(\[].*?[\)\]]` | *（空欄）* | `映画タイトル (2024).mp4` | `映画タイトル.mp4` |
| `\.JPG$` | `.jpg` | `IMG_001.JPG` | `IMG_001.jpg` |

> **ポイント**
> - 置換欄を**空欄**にするとマッチした部分を削除できます。
> - `$1`・`$2`・… は左から順に `()` グループに対応します。
> - **正規表現**チェックをオンにしないと `$1` が文字通り `$1` として扱われます。

## ソースからビルド

**必要なもの：** .NET 10 SDK

```bash
git clone https://github.com/umineko73/RenText.git
cd RenText
dotnet run
```

**配布用ビルド（シングルファイル・framework-dependent）：**

```bash
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true --output ./publish
```
