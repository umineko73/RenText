# 変更履歴

このプロジェクトの主要な変更はすべてこのファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠し、
バージョン管理は [セマンティック バージョニング](https://semver.org/lang/ja/) に従います。

## [Unreleased]

## [0.1.2] - 2026-05-24

### 追加
- `F5` キーでフォルダツリーとファイル一覧を手動リフレッシュ可能に
- フォルダツリーペインとプレビューペインの幅を `settings.json` に保存し、次回起動時に復元するよう対応

### 修正
- サブディレクトリをリネームしてもフォルダツリーが古い名前のまま更新されなかった問題を修正
- Windows の `ReadOnly` ファイル属性（カスタムフォルダアイコン用）が付いたディレクトリが読み取り専用と誤判定され、リネーム対象から除外されていた問題を修正

### 変更
- .NET 9 から .NET 10 へ移行
- その他微修正

## [0.1.1] - 2026-05-24

### 追加
- ファイルペインにサブディレクトリも表示されるようになり、ファイルと同様にリネーム可能
- ディレクトリには 📁 アイコンを表示。アクティブなソート列に関わらず常にファイルより上に表示
- 自動更新がカレントフォルダ内のサブディレクトリの作成・削除・リネームも検知するよう対応

### 変更
- フォルダツリーペインの初期幅を 220 px から 280 px に拡大
- ウィンドウサイズと最大化状態を `settings.json` に保存し、次回起動時に復元するよう対応

## [0.1.0] - 2026-05-21

### 追加

#### コア編集機能
- テキストエディタ感覚のファイル名編集 — 1行1ファイルで直接編集
- リアルタイムプレビューペイン — 変更前後を `変更前 → 変更後` 形式で一覧表示
- 完全ロールバック付き保存（`Ctrl+S`）— 1件でもエラーがあればすべての変更をアトミックに取り消し
- サイクル対応リネーム — A→B / B→A の入れ替えなど循環するリネームを、トポロジカルソートと一時ファイルを使って正しく処理

#### 検索・置換
- VS Code スタイルの検索・置換パネル（`Ctrl+H`）
- 正規表現サポート
- 大文字小文字区別オプション
- 入力しながらプレビューペインにリアルタイム反映

#### ファイル一覧
- ファイル名 / 更新日時 / 拡張子 / サイズ でのソート（昇順・降順）
- 表示形式切替：ファイル名のみ / 更新日時付き / サイズ付き / 日時＋サイズ
- `↑` `↓` `PageUp` `PageDown` によるキーボードナビゲーション（カーソル列位置を維持）
- ステータスバーに行番号表示（`Ln N`）

#### フォルダツリー
- 遅延読み込み対応フォルダツリー（ドライブアイコン付き）
- ノード選択・展開時の水平方向への自動スクロールを抑制

#### 自動更新
- 外部アプリによるファイルの追加・削除・リネームを検知して自動更新
- 表示中フォルダ自体のリネーム・削除にも追従（アドレスバーとツリーノードを自動更新）
- 未保存の編集がある場合は自動更新を一時停止（ステータスバーに通知表示）

#### UI / UX
- ダーク / ライトモード切替
- 日本語 / 英語の言語切替（設定はセッションをまたいで保持）
- 隠しファイル・システムファイルの表示切替
- 保存成功トースト通知（1秒間表示）
- 行単位のエラーハイライトとツールチップによるエラーメッセージ表示
- テーマと言語設定の永続化（`settings.json`）

#### 配布
- Windows x64 シングルファイルビルド（framework-dependent）— [.NET 9 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/9.0) が必要
